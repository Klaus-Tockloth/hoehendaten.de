Eigene Objekte: GPX-, KML- oder GeoJSON-Daten importieren

Diese Ebene ermöglicht es Ihnen, eigene geographische Objekte wie Punkte oder Linien auf der Karte darzustellen. Durch den Import von Dateien in gängigen Formaten wie GPX, KML oder GeoJSON können Sie individuelle Daten wie Wanderrouten, Standorte oder Gebietsgrenzen visualisieren und analysieren.

Sie können verschiedene Parameter anpassen, um die Darstellung zu steuern:

  • Unterstützte Formate: Für den Import stehen Ihnen die folgenden weitverbreiteten Datenformate zur Verfügung:
    • GPX
    • KML
    • GeoJSON
  • Standardstil: In der Konfiguration der Ebene können Sie einen allgemeinen Darstellungsstil (z. B. Farbe, Linienstärke) festlegen, der automatisch auf alle neu importierten Objekte angewendet wird.
  • Individuelle Anpassung: Jedes importierte Objekt kann nachträglich individuell gestaltet werden. Wählen Sie dazu das gewünschte Objekt in der Objektübersicht aus, um dessen Stil (z. B. Farbe, Linienart) gezielt zu verändern.
  • Interaktion mit dem Kartenbild: Die Ebene kann durch die Auswahl eines Mischmodus und/oder durch Filter mit darunterliegenden Kartenebenen interagieren. Dadurch lassen sich visuelle Effekte erzielen, die die Lesbarkeit oder die ästhetische Integration der Objekte verbessern.
